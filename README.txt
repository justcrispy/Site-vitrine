Ce site est un site mirroir, il faut télécharger l'application pour suivre les étapes du site dans le logiciel.


Voici le site qui est disponible dans le fichier Accueil.html

- Pour retourner à l'accueil : Appuyer sur "NFT FACTORY"

- Aller voir les exemples NFT : Appuyer sur "Galerie"

- Voir le fonctionnement du site : Appuyer sur "Comment ça fonctionne ?"

- Voir les personnes qui ont contribués au projet : Appyuyer sur "Qui nous-sommes ?"

- Commencer la génération des images NFT : Appuyer sur "Commencer"


Etape 1 : Choisir l'arrière plan de l'image NFT.
Etape 2 : Choisir la forme de visage deu personnage.
Etape 3 : Choisir la couleur de peau du personnage.
Etape 4 : Télécharger le logiciel pour générer les images. 

Une fois téléchargé, suivre les étapes ci-dessous (1 à 3) avec le format du site en aide.
